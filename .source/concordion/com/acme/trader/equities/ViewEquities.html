The countries file includes GB and US

When

The list of equities are created, based on the country codes specified in the external countries file (for example countries.csv)

The equities as a resource is queried with a HTTP GET on the URL... 

/trader/equities

with a "accept" mime-type of... 

application/json

Then

  • The list of equities are returned for country codes GB and US
  • Any equities associated with country code RU or ZI should not be retrieved
