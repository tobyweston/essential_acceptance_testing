Archive Corporate Actions via REST - Ignoring Fields

Given

The system receives the following corporate actions from EPIC which is not archived

Event ID Type Ex Date Record Date Payment Date Gross Dividend Net Dividend Withholding Tax Rate ISIN SEDOL CUSIP Currency Source System Code
4526892 CD 2010-06-07 00:00:00.000 GMT+00:00 2010-06-10 00:00:00.000 GMT+00:00 2011-07-01 00:00:00.000 GMT+00:00 0.3369 0.7885 GB0031129355 B1YFR51 GBP APM,GLB,SES,SSE,TEM

When

The user archives the corporate action with id 4526892 via REST

A HTTP PUT request is made to archive the corporate action resource on the URL 

/trader/corporate_actions
        {
          "id" : "4526892",
          "type" : "DOR",
          "exDate" : "2011-03-21",
          "recordDate" : "2011-03-13",
          "paymentDate" : "2011-04-01",
          "grossDividend" : 0.75,
          "netDividend" : 0.621,
          "isin" : "US78402X1072",
          "sedol" : "B1HK6N4",
          "currency" : "GBP",
          "archived" : true
        }

Note: all fields but "id" and "archived" will be ignored.

Then

The corporate action 4526892 is archived and the HTTP response includes the status 200 (OK).
