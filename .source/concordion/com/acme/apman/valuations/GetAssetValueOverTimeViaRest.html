When

The valuations for assets in the default portfolio is made via REST

A HTTP GET on the following URL is made A HTTP GET on the following URL is made 

/apman/valuations/portfolio/total

Then

The system calculates the value of each asset in the portfolio currency for every day and returns a JSON representation. In order to calculate the value of the portfolio for a given date, it must be possible to value every asset on that date. If a single asset can't be valued for a given day, the value of the portfolio can't be calculated for that day.

To calculate the asset value for a given day, we use position * closing price in local currency * portfolio currency

Example

{
  "valuations" : [
    {
      "effectiveDate" : "2011-08-10",
      "portfolioCurrencyValue" : 520934.44
    }
  ]
}
