Sell LEPO / Buy Asset Booking in GR

Having previously created a LEPO instrument in GR and tallied onto it as a result of concluding the Sell Asset / Buy LEPO workflow, the LEPO will be exercised on expiry by executing or cancelling the Sell LEPO / Buy Asset workflow.

When

APMan receives a request to execute an Sell LEPO / Buy Asset transaction. A request to exercise the LEPO is made to GR (via Dragon) including a dividend cash flow adjustment.

Example

Transaction Identifier GR Symbol Currency Executed Quantity Executed Price Gross Dividend Tax Rate Strategy Risk Class Family SubBook Trade Date Exercise Date Settlement Date
7582348b-37e1-2345-9431-3545645621a1 CG=P GBP 1000.6 100.2 0.311832 0.33 MY_STRAT RISKY RRD SUB_1 2011-06-29 2011-06-30 2011-04-01

The tally premium is calculated as the gross dividend rate * (1 - tax rate)

The exercise date on the Sell LEPO / Buy Asset transaction is used as the expiration date.

Then

A valid JSON message is sent to Dragon representing a negative tally onto an existing LEPO (as referenced by the symbol) via a HTTP POST.
