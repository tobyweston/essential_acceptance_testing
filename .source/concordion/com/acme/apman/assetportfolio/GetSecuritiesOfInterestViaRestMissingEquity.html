Get Securities of Interest via REST with Missing Equities

Given

Assets exist in the system but some equities are not available in the repository,

When

A HTTP GET request is made to the asset soi web service

A HTTP GET request is made on the URL... 

/apman/assets/soi

with a "accept" mime-type of... 

text/tab-separated-values

Then

A valid EPIC securities of interest tab separated file is returned with no values for the equities that are not longer present.
